Tips

  1. Hide and Show Deleted Items: Click the Show Deleted Items button at the bottom right corner of the window to show or hide deleted items. If shown, deleted items appear in red. When deleted items are shown, the Show Deleted Items button has a red label that says shows the number of deleted rows and columns.

    Note: If you show deleted items and there aren't any, then no items will be red, but the label that shows the number of deleted items will still appear.

  2. Open Recent Lists: Use the "Open Recent" item in the "File" menu to open Delectus files that you have used recently.

  3. Rearrange Columns: You can drag a column to a new place in the window. Simply grab the column's label and drag it to the new place yu want it to be. Delectus will remember the order you choose.

    You can also change the width of a column by dragging its right edge. Grab the right edge of the column's label and drag it to resize the column. Once again, Delectus will remember the size you choose.

  4. Change the Font Size: If the font in the Delectus window is too large or too small, you can change it using the items in the Font menu. Delectus remembers the last font size you chose.
